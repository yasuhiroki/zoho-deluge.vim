# zoho-deluge.vim

Vim syntax highlighting for Zoho Deluge.

## Installation

Using [vim-plug](https://github.com/junegunn/vim-plug):

```vim
Plug 'yasuhiroki/zoho-deluge.vim'
```

## Features

- Basic Deluge syntax highlighting (similar to JavaScript).
- Special syntax highlighting for Deluge-specific tasks such as `invokeUrl`.
